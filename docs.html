Install CLI

Requirements

  • Windows 10, Mac OSX, Linux
  • Node.js v12 or higher

Install

Mac (Homebrew)

$ brew install Beroski-CLI

NPM

$ npm install Beroski-CLI

YARN

$ yarn install Beroski-CLI
